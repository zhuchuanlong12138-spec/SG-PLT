================================================================================
测试环境
================================================================================
测试用板:
---------------------
HC32L110_STK

辅助工具:
---------------------
示波器
电源

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）打开工程编译
2）运行程序，发送一字节0x9f给spi_slave
3）示波器观测P03、P23、P24、P25可以观察到相应的波形
4) 初始化时，将P32电平置低，收发完成以后，会将P32电平置高；rx_buf收到3 bytes返回值{0x1,0x2,0x15}

================================================================================
注意
================================================================================
1）SPI 相应的引脚设置
   P03：CS
   P23：MISO
   P24：MOSI
   P25：SCK
